installing chrome plugin:
1. chrome://extensions/
2. check 'Developer mode'
3. 'Load unpacked extension...'
4. select 'chrome' directory

deploying app to heroku:

```
git push heroku master
```

running app:

```
heroku open
```
or https://mysterious-tundra-16826.herokuapp.com/

running tests:

```
jasmine
```
